How to compile this software:
This will compile on Linux. Start by unzipping the contents of the zip folder.
Open a console and change the path to the folder in which the source files were
unzipped. Running the following command will compile the code for you:

	make

The calculator can then be run by typing:

	./calc

There is a short help text if you type:

	./calc help

You must enter expressions without spaces.
Expressions must end with a semicolon ;
